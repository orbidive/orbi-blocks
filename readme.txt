== Description ==

=== Gutenberg Blocks ===

Orbi Blocks is a collection of WordPress page-building Gutenberg blocks that offer unlimited possibilities for designing webpages. It allows you to create professional and elegant websites using the new WordPress editor, Gutenberg, without needing to write any code. Orbi Blocks provides a page-building experience with advanced capabilities and graceful designs that have not been seen before in any other page builder. It can be used with any theme to create any design.

Orbi Blocks enhances the Gutenberg editor by providing a wide range of attractive blocks that can be easily customized to create and design web pages with just a few clicks. This allows for more flexibility in content creation and design, reducing the constraints of other page builders and offering unlimited possibilities for building simple to complex WordPress websites using the Gutenberg editor. It has been developed with all the necessary customization features for blocks to achieve any layout and design for web page content with the Gutenberg block editor. Orbi Blocks simplifies the process of designing any type of web page by enabling users to simply click and drag and drop elements.

Orbi Blocks is a highly optimized lightweight WordPress plugin, featuring the introduction of Internal or External CSS files for dynamic CSS optimization, aimed at maximizing performance. With Highly Typography Feature.

## Plugin Development

This plugin uses npm and webpack to build the production files. To build the files yourself, follow these steps:

### Requirements

Node.js >= 20.x

### Install

Clone the repository:

```bash
git clone https://github.com/orbidive/orbi-blocks.git
```

Navigate to the plugin directory: "cd orbi-blocks"

Install dependencies:

```bash
npm install
```

or

```bash
npm install --force
```

## Development

Orbi Blocks using @wordpress/scripts to handle compiling assets and watching for changes.

See the [Official Gutenberg Developer Documentation](https://developer.wordpress.org/block-editor/#develop-for-the-block-editor) for extensive tutorials, documentation, and API references.

### NPM Scripts

Use the following commands to get started:

Watch for changes:

```bash
npm start
```

Build a production version of Orbi Blocks:

```bash
npm run build
```
